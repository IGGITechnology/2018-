**a.与Git同类型的工具有？**  
   
分布式版本控制系统除了Git以及促使Git诞生的bitkeeper外，还有类似Git的mercurial和bazaar等。与其同类型的工具则有svn和cvs等。  
****  
**b.Git分了哪几个区？**  
  
Git大致分为三个区：  
1.工作区，就是我们可见的目录。  
2.暂存区，为工作区和版本库之间增加一个过渡。  
3.版本库，存放各个版本。
